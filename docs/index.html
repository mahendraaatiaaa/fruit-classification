Image Requirements:

  • The background color of the image should be plain white.
  • The fruit image should not be a 2D or 3D illustration; it should be a real photo of the fruit.
  • The image should contain only one fruit, positioned in the center.
  • The fruit should have typical colors and contents that are common for that type of fruit.
  • The fruit should not be rotten or show any signs of spoilage.
